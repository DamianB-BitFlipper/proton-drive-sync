Proton Drive Sync

Sync local directories to Proton Drive cloud storage.

GitHub Repository

macOS (Homebrew)

brew tap DamianB-BitFlipper/tap
brew install proton-drive-sync
proton-drive-sync setup

Debian / Ubuntu

Download the .deb package from GitHub Releases: 

# Download the .deb for your architecture (amd64 or arm64)
# Then install with:
sudo apt install ./proton-drive-sync_*.deb

# Run setup wizard
proton-drive-sync setup

Fedora / RHEL / CentOS

Download the .rpm package from GitHub Releases: 

# Download the .rpm for your architecture (x86_64 or aarch64)
# Then install with:
sudo dnf install ./proton-drive-sync-*.rpm

# Run setup wizard
proton-drive-sync setup

Other Linux

Download the Linux tarball from GitHub Releases: 

# Extract and install manually
tar -xzf proton-drive-sync-linux-x64.tar.gz
sudo mv proton-drive-sync /usr/local/bin/

# Run setup wizard
proton-drive-sync setup

Windows

Download the .zip from GitHub Releases, extract, and add to your PATH.

Post-Installation

After installing, run the setup wizard to authenticate and configure the service:

proton-drive-sync setup
Note: The setup wizard will guide you through Proton authentication and optionally install a system service for automatic syncing.
